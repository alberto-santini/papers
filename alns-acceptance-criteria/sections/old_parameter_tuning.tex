\section{Parameter Tuning}

\subsection{Parameters kept fixed}
A number of parameters are kept at fixed values. The following sub-sections describes the value of those parameters.

\begin{itemize}
\item Number of nodes to destroy$\bar{\omega}^{+}=50,$ $\omega^{+}=0.4,\bar{\omega}^{-}=10,\omega^{-}=0.1$ 
\item Relatedness destroy method: $pRel=5$
\item Historical node-pair destroy: $p^{hist}=5$
\end{itemize}

\paragraph{Repair methods: }
\begin{itemize}
\item Regret repair: $p^{regret}=1.5$ (for the stochastic version of the
method)
\end{itemize}

\paragraph{Local improvement methods}
\begin{itemize}
\item $p^{2-opt}=0.1$ 
\end{itemize}
